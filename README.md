MIC - made in china
----

> 使用 vue 2 + webpack 2 构建的 UI 组件库.
> @by Richie

CLI
----

安装

    npm install

本地开发

    npm run dev

本地打包

    npm run build

本地打包，包含 CSS / HTML

    npm run pack
